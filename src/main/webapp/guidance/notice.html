Box Plot Guidance

注意一：

导入数据库的表名、字段名都必须为英文。且字段必须符合某个功能的进一步需求，详情看具体指导。

注意二：

系统运行前，必须开启MySQL服务，点击“开始”→“运行”→输入“cmd”→输入“net start mysql”

注意三：

若数据库某些属性获取失败，请检查注意二，找到ApplicationContext.xml中的datasource，检查数据库名、接口、用户名与密码是否正确。
